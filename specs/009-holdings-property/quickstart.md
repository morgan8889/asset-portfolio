# Quickstart: Property & Holdings

## Overview
Manage your complete portfolio by adding Real Estate assets, tracking rental income, and monitoring total net worth in one unified view.

## Usage

### Adding a Property
1. Navigate to **Holdings**.
2. Click **"Add Holding"**.
3. Select **"Real Estate"** tab.
4. Enter property details:
   - **Name**: e.g., "Downtown Condo"
   - **Current Value**: Estimated market value.
   - **Purchase Price**: Original cost.
   - **Ownership %**: Your share (default 100%).

### Tracking Rental Income
1. In the Add/Edit Property form, toggle **"Rental Property"**.
2. Enter the **Monthly Rent**.
3. Save. The system now calculates **Annual Yield** based on the current value.

### Updating Property Value
1. Locate the property in the **Holdings List**.
2. Click the **"Update Value"** button (pencil icon).
3. Enter the new estimated value.
4. The portfolio total updates immediately, and a price history point is recorded.

### Filtering Holdings
1. On the Holdings page, use the **"Filter Type"** dropdown.
2. Select **"Real Estate"** to see only your properties.
3. Select **"All"** to return to the full list.

## Implementation Details

- **Net Value**: Calculated as `Current Value * Ownership %`.
- **Manual Updates**: Unlike stocks, property values don't update automatically. You control the valuation.
- **Privacy**: All property data is stored locally on your device.
